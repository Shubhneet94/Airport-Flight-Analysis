# Airport & Flight Analysis using SQL and Power BI

## Project Overview
This project analyzes airline operations, passenger traffic, airport performance, freight movement, and route efficiency using SQL and Power BI.

## Tools Used
- MySQL
- Power BI
- SQL
- DAX

## Project Features
- Database normalization
- ETL and data cleaning
- Route analysis
- Airport performance analysis
- Passenger and freight KPIs
- Interactive Power BI dashboard

## Database Tables
- Airline
- Airport
- Flight
- FlightMetrics
- City

## SQL Concepts Used
- Joins
- CTEs
- Views
- Stored Procedures
- Aggregate Functions
- Window Functions
- Indexing

## Power BI Dashboard
Dashboard includes:
- Total Passengers
- Total Flights
- Top Airlines
- Top Routes
- Airport Analysis
- State-wise Traffic Map

## Key Insights
- High-traffic routes identified
- Top-performing airports analyzed
- Freight-heavy corridors identified
- Passenger distribution across states analyzed
